# ShadowDeploy

Bounded shadow-testing harness for AWS — mirrors sampled production traffic to a
new version, diffs outputs **without serving them**, and reports divergence
before promotion.

See [`docs/README.md`](docs/README.md) for full documentation, and
[`SPRINT_TRACKER.md`](SPRINT_TRACKER.md) for the build history.
